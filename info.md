配置例子,填写绑定小爱音箱的用户账号信息：
 
```
hello_miai:
  miid: '13123456789'
  password: 'password'
```

调用服务，如果你有多个音箱，miai_num代表你的音箱id。
```
wait_time: 0
miai_num: 0
message: 你好天睿tera，我不是小爱。
```
